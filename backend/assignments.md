# Prerequisits
* Java JDK 8
* Maven 3
* Docker or MongoDB
* IDE

# Getting started
* git clone https://github.com/erwindeg/ultimate-async-stack
* cd backend
* mvn clean install
* cd backend/db
* docker-compose up (to start the MongoDB with docker and import the data)
* Run MainWebVerticle from your IDE
* go to http://localhost:8080/api/movies
